## Configuração

1. Clone o repositório
2. Copie `.env.example` para `.env`
3. Edite `.env` e adicione sua chave de API OpenAI
4. Instale as dependências: `pip install -r requirements.txt`
5. Execute o aplicativo: `python app.py`
